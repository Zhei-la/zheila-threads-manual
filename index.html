PART 02 — Threads 토큰 발급
REQUIRED

자동 발행을 위한 토큰 발급

예약 발행, 자동 발행을 사용하려면 Threads 장기 액세스 토큰이 필요해요.
한 번만 발급받으면 60일간 유효해요.

무료 10분 소요 60일 유효
1단계

Meta 개발자 콘솔 접속 및 로그인

https://developers.facebook.com
1
오른쪽 상단 로그인 클릭Threads와 연결된 Facebook 계정으로 로그인해요
!
회원가입이 안 될 경우페이스북 오류로 번호 인증이 안 되면 잠시 기다렸다 재시도하거나 다른 번호로 가입해보세요. 이미 페이스북 계정이 있다면 그 계정으로 바로 로그인하면 돼요.
2
로그인 후 오른쪽 위 내 앱 클릭상단 메뉴에서 내 앱을 클릭해요
2단계

앱 만들기

1
오른쪽 위 앱 만들기 클릭파란 버튼을 눌러요
2
앱 이름 입력 후 이용사례 추가이용사례 추가에서 Threads API 액세스 클릭 → 다음
3
비즈니스 / 요구사항 패스비즈니스 포트폴리오, 요구사항은 그냥 건너뛰고 오른쪽 아래 앱 만들기 클릭
3단계

앱 시크릿 코드 복사

1
앱 설정 → 기본 설정 클릭왼쪽 메뉴에서 앱 설정 → 기본 설정으로 이동해요
2
Threads 앱 시크릿 코드 보기 → 복사아래로 스크롤해서 Threads 항목을 찾아요
!
메모장에 저장 (절대 공유 금지!)앱 시크릿은 절대 다른 사람에게 보여주면 안 돼요
4단계

테스터 추가

1
앱 역할 → 역할 클릭왼쪽 메뉴에서 앱 역할 → 역할을 클릭해요
2
오른쪽 위 사람 추가 클릭역할 선택 화면이 나와요
!
맨 밑에 Threads 테스터 선택아이디를 직접 입력하고 선택 후 추가 클릭
⚠️ 복사 붙여넣기 안 됨! 직접 손으로 쳐야 해요
5단계

Threads 앱에서 초대 수락

1
Threads 앱 접속스마트폰 Threads 앱을 열어요
2
설정 → 계정 → 웹사이트 권한4개 메뉴 중 맨 마지막 초대 클릭
3
승인 클릭 후 Meta로 돌아가기승인하면 대기중이 사라지고 테스터로 등록돼요
6단계

단기 토큰 발급

1
테스트 클릭 → 그래프 API 탐색기 열기Meta 개발자 콘솔에서 테스트 버튼을 눌러요
2
왼쪽 위 GET 옆에 Facebook → Threads로 변경드롭다운에서 Threads로 바꿔줘요
3
Generate Access Token 클릭오른쪽 액세스 토큰에서 클릭 또는 "액세스 토큰 생성" 버튼
!
아이디로 로그인하겠습니까? → 확인확인 누르면 단기 토큰 발급. 복사해서 메모장에 저장!
⚠️ 1시간만 유효! 바로 7단계로!
7단계

장기 토큰 발급 (60일짜리)

3단계 앱 시크릿 + 6단계 단기 토큰을 아래 URL에 넣어요

⚠️ 주의사항 (안 지키면 오류 남!)
· 띄어쓰기 절대 없어야 함
· 한글 텍스트가 남아있으면 안 됨
· 전부 영문+숫자로만 구성돼야 함
· 단기 토큰은 발급 후 1시간 이내 사용
https://graph.threads.net/access_token?grant_type=th_exchange_token&client_secret=앱시크릿&access_token=단기토큰
1
메모장에 붙여넣고 교체 후 브라우저 주소창 입력앱시크릿, 단기토큰 값으로 교체. 한글 없어야 정상!
2
JSON 응답에서 access_token 값만 복사따옴표 안쪽 값만 복사 → 이게 60일짜리 장기 토큰이에요
{
  "access_token": "THQCl...여기가 장기 토큰...XYZ",
  "token_type": "bearer",
  "expires_in": 5183944
}
👆 access_token 의 값(따옴표 안쪽)만 복사!
8단계

앱에 토큰 등록

1
👤 계정 관리 메뉴 클릭왼쪽 사이드바에서 계정 관리를 눌러요
2
계정 이름 입력구분하기 쉬운 이름 — 예: 메인 계정, 쿠팡 계정
3
장기 액세스 토큰 붙여넣기 → + 계정 추가 클릭완료! 이제 자동 발행을 쓸 수 있어요 🎉
💡 참고 — 다른 계정도 추가하고 싶다면 4단계(테스터 추가)부터 다시 진행하면 돼요!
