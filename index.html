예약 발행
01

가입하기

1
사이트 접속 관리자에게 받은 주소로 접속해요
2
가입 탭 클릭 상단 가입 버튼을 눌러요
3
초대코드 입력 관리자에게 받은 초대코드를 입력해요
4
이름 / 닉네임 / 비밀번호 입력 실명과 사용할 닉네임을 입력해요
5
가입하기 클릭 관리자 승인 후 로그인 가능해요
02

글 생성하기
(토큰 없이 가능)

1
✏️ 글 작성 메뉴 클릭 왼쪽 사이드바에서 글 작성을 눌러요
2
주제 입력 제품명이나 주제를 입력해요
예: 에어프라이어, 강아지 간식, AI 뉴스
3
말투 선택 리스트형 정보성 리뷰형
일상 공감형 스토리형 쿠팡
4
🪄 글 생성 클릭 AI가 자동으로 한국어 SNS 글을 써줘요
5
📋 복사 클릭 Threads 앱에 붙여넣기해서 발행해요
TIP

사진 넣으면 더 좋은 글이 나와요

글 작성 상단에서 사진을 먼저 업로드하면
AI가 사진을 분석해서 더 맞춤화된 글을 써줘요.

PART 02 — Threads 토큰 발급
REQUIRED

자동 발행을 위한 토큰 발급

예약 발행, 자동 발행을 사용하려면
Threads 장기 액세스 토큰이 필요해요.
한 번만 발급받으면 60일간 유효해요.

무료 10분 소요 60일 유효
1단계

Meta 개발자 콘솔 접속

1
브라우저에서 접속 주소창에 입력해요
클릭하면 복사돼요
https://developers.facebook.com
바로 접속하기 →
2
오른쪽 상단 로그인 Facebook 계정으로 로그인해요
(Instagram/Threads와 연결된 계정)
3
내 앱 클릭 상단 메뉴에서 내 앱을 클릭해요
2단계

앱 만들기

1
앱 만들기 버튼 클릭 오른쪽 상단 파란 버튼을 눌러요
2
사용 사례 선택 기타 → 다음 클릭해요
3
앱 유형 선택 소비자 선택 → 다음 클릭
4
앱 이름 입력 아무 이름이나 입력해요
예: MyThreadsApp
5
앱 만들기 클릭 비즈니스 포트폴리오는 비워도 돼요
3단계

Threads API 설정

1
왼쪽 메뉴 → 제품 추가 앱 대시보드 왼쪽에서 제품 추가를 눌러요
2
Threads API 찾기 스크롤해서 Threads API를 찾아요
3
설정 클릭 Threads API 옆 설정 버튼을 눌러요
4
왼쪽 메뉴 → Threads API → 설정 Threads API 설정 페이지로 이동해요
4단계

테스터 추가

1
사용자 토큰 생성기 섹션 찾기 설정 페이지를 아래로 스크롤해요
2
Threads 테스터 추가 본인의 Threads 계정 아이디를 입력해요
예: @my_threads_id
3
초대 수락 Threads 앱 → 설정 → 개발자 도구
→ 앱 초대 수락 클릭
5단계

단기 토큰 발급

1
사용자 토큰 생성기로 이동 설정 페이지 하단 섹션이에요
2
액세스 토큰 생성하기 클릭 파란 버튼을 눌러요
3
권한 허용 팝업에서 계속, 허용 클릭해요
4
단기 토큰 복사 생성된 토큰을 메모장에 붙여두세요
1시간만 유효해요 — 빨리 다음 단계로!
6단계

장기 토큰으로 교환
60일짜리로 바꾸기

앱 설정에서 앱 시크릿을 복사한 후
아래 URL을 브라우저 주소창에 입력해요

왼쪽 메뉴 → 앱 설정 → 기본 설정 → 앱 시크릿 표시 클릭 후 복사

아래를 클릭하면 복사돼요. 앱시크릿, 단기토큰 부분만 교체해서 주소창에 붙여넣기!
https://graph.threads.net/access_token
?grant_type=th_exchange_token
&client_secret=앱시크릿붙여넣기
&access_token=단기토큰붙여넣기
1
URL 복사 후 앱시크릿, 단기토큰 교체 앱시크릿 → 방금 복사한 시크릿으로
단기토큰 → 방금 발급받은 토큰으로
2
브라우저 주소창에 붙여넣고 Enter JSON 응답이 나와요
3
access_token 값 복사 응답 중 "access_token":"여기값" 을 복사해요
이게 60일짜리 장기 토큰이에요!
7단계

앱에 토큰 등록

1
👤 계정 관리 메뉴 클릭 왼쪽 사이드바에서 계정 관리를 눌러요
2
계정 이름 입력 구분하기 쉬운 이름을 입력해요
예: 메인 계정, 쿠팡 계정
3
장기 액세스 토큰 붙여넣기 방금 복사한 60일짜리 토큰을 붙여넣어요
4
+ 계정 추가 클릭 완료! 이제 자동 발행을 쓸 수 있어요
